# SISTEMA ESG

Aplicação ESG para treinamento pessoal

## Setup e Uso

### Execução / Local H2 Database dev

#### Terminal

```sh
./mvnw spring-boot:run -Dspring-boot.run.profiles=dev
```
### Execução / Oracle Database prd

#### Terminal

```sh
./mvnw spring-boot:run -Dspring-boot.run.profiles=prd
```
